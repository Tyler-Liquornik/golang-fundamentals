Functions are first-class values in Go. We can store them in variables and pass them around easily, similar to JavaScript.

Let's start with some code.

`main.go`:
```go
package main

import "fmt"

func main() {
	nums := []int{1, 2, 3, 4, 5}
	doubleNumbers(nums)
	fmt.Println(nums)
}

func doubleNumbers(nums []int) {
	for i := range nums {
		double(&nums[i])
	}
}

func double(number *int) {
	*number *= 2
}
```

Now consider we want to add a function `triple` and `tripleNumbers`. And then maybe `subtractOneFromNumbers` and so on in the future. At this point, we may as well generalize to `transformNumbers(nums *[]int, transform func(*int))`.

`main.go`:
```go
package main

import "fmt"

type transformFunction func(*int)

func main() {
	nums := []int{1, 2, 3, 4, 5}
	transformNumbers(nums, double)
	transformNumbers(nums, subtractOne)
	fmt.Println(nums)
}

func transformNumbers(nums []int, transform transformFunction) {
	for i := range nums {
		transform(&nums[i])
	}
}

func double(number *int) {
	*number *= 2
}

func subtractOne(number *int) {
	*number -= 1
}
```

> It’s common to define a type for function signatures when they’re reused or become complex.

Functions can also return functions as values, also similar to Javascript. Let's write `getTrasnformerFunction` that returns us from a string key the actual function. Think here like passing in the name of a Redux action to dispatch and then switching on that key to then return the action.

`main.go`:
```go
package main

import "fmt"

type transformFunction func(*int)

func main() {
	nums := []int{1, 2, 3, 4, 5}
	transformNumbers(nums, getTransformerFunction("double"))
	transformNumbers(nums, getTransformerFunction("subtractOne"))
	fmt.Println(nums)
}

func transformNumbers(nums []int, transform transformFunction) {
	for i := range nums {
		transform(&nums[i])
	}
}

func getTransformerFunction(name string) transformFunction {
	switch name {
	case "double":
		return double
	case "subtractOne":
		return subtractOne
	default:
		return nil
	}
}

func double(number *int) {
	*number *= 2
}

func subtractOne(number *int) {
	*number -= 1
}
```
