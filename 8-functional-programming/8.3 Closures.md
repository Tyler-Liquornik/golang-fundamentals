Starting from the code in 8.2, consider we add some code and get to:

`main.go`:
```go
package main

import "fmt"

func main() {
	numbers := []int{1, 2, 3}
	
	doubleTransformer := createTransformer(2)
	transformed := transformNumbers(&numbers, doubleTransformer)

	fmt.Println(transformed)
}

func transformNumbers(numbers *[]int, transform func(int) int) []int {
	dNumbers := []int{}

	for _, val := range *numbers {
		dNumbers = append(dNumbers, transform(val))
	}

	return dNumbers
}

func createTransformer(factor int) func(int) int {
	return func(num int) int {
		return num * factor
	}
}
```

The anonymous function inside `createTransformer` has a closure which captures the value of `factor` from its surrounding scope, even though it’s not defined within the function itself. This is similar to how closures work in JavaScript. This is similar to how closures work in JavaScript.
