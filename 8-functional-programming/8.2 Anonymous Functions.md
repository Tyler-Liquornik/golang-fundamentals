How can we fill in `???` to double the numbers without defining a new function with `func function(...)` if we don't intend to use it anywhere else. We can use an **anonymous function**, just like we would in Java with a Lambda function.

`main.go`:
```go
package main

import "fmt"

func main() {
	numbers := []int{1, 2, 3}

	transformed := transformNumbers(&numbers, func(num int) int {
		return num * 2
	})

	fmt.Println(transformed)
}

func transformNumbers(numbers *[]int, transform func(int) int) []int {
	dNumbers := []int{}

	for _, val := range *numbers {
		dNumbers = append(dNumbers, transform(val))
	}

	return dNumbers
}
```
