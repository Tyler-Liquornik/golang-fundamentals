Let's really dissect what's going on with our Hello World program:

`app.go`:
```go
package main  
  
import "fmt"  
  
func main() {  
    fmt.Print("Hello World")  
}
```


Let's first take a look at `func main()`. In that function, we call the **function** `Print` to write the **String value** "Hello World" to the standard output stream `stdout`. We are calling the function `Print` from its **package** `fmt`, that we imported with `import "fmt"`.  

> You must only have one `main` function inside the `main` package. You may not however need a `main` function, for example if you are exposing packages for others to import
