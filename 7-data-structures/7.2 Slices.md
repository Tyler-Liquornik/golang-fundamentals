A **slice** is a reference to a contiguous segment of an array. Slice an existing array (or another slice) `arr = [n]T` like `slice := arr[i:j}`, `0 <= i <= j <=n`. This slice includes all elements from `i` inclusive to `j` exclusive.

Declaring using the same syntax as an array without specifying a size `n` in `[n]T` creates an empty slice like `slice := []T{}`.

`lists.go`:
```go
package main

import "fmt"

func main() {
	var productNames [4]string
	productNames = [4]string{"Apple"}
	productNames[2] = "Kiwi"
	fmt.Println(productNames)

	prices := [4]float64{10.99, 9.99, 45.99, 20.0}
	fmt.Println(prices)
	fmt.Println(prices[2])

	featuredPrices := prices[1:3]
	fmt.Println(featuredPrices)
}
```

Result:
```sh
[Apple  Kiwi ]
[10.99 9.99 45.99 20]
45.99
[9.99 45.99] # Only index 1, 2, (3 was exlcuded)
```

We can also slice `arr[:j]` or `arr[i:]` which implies `i = 0` or `j = n` respectively. We can even write `arr[:]` to get a slice of the whole array.

`lists.go`:
```go
package main

import "fmt"

func main() {
	var productNames [4]string
	productNames = [4]string{"Apple"}
	productNames[2] = "Kiwi"
	fmt.Println(productNames)

	prices := [4]float64{10.99, 9.99, 45.99, 20.0}
	fmt.Println(prices)
	fmt.Println(prices[2])

	featuredPrices := prices[1:]
	highlightedPrices := featuredPrices[:1]
	fmt.Println(featuredPrices)
	fmt.Println(highlightedPrices)
}
```

Result:
```sh
[Apple  Kiwi ]
[10.99 9.99 45.99 20]
45.99
[9.99 45.99 20]
[9.99]
```

An important thing to know about slices is that they are built on top of arrays and share memory with their underlying array, so if you R/W on them you R/W on the array. This also means if you take a slice of a slice, you can expand the second slice beyond the first slice, within the original array. The constraint is that you can only expand to the right, because a slice internally is a set of variables `(pointer to first element, length, capacity)`. Indices `i` and `j` in a slice always refer to indices for the underlying array.

> When you create an empty slice (e.g., `[]T{}`), Go still creates an underlying empty array behind the scenes, but it isn’t directly accessible. If you declare but don't assign an empty slice, it will be `nil`, the underlying array isn't created until an assignment is made.

`lists.go`:
```go
package main

import "fmt"

func main() {
	var productNames [4]string
	productNames = [4]string{"Apple"}
	productNames[2] = "Kiwi"
	fmt.Println(productNames)

	prices := [4]float64{10.99, 9.99, 45.99, 20.0}
	fmt.Println(prices)
	fmt.Println(prices[2])

	featuredPrices := prices[1:]
	featuredPrices[0] = 199.99
	highlightedPrices := featuredPrices[:1]
	fmt.Println(highlightedPrices)
	fmt.Println(prices)
}
```

Result:
```sh
[Apple  Kiwi ]
[10.99 9.99 45.99 20]
45.99
[199.99]
[10.99 199.99 45.99 20] # editing featuredPrices overwrote prices since featuredPrices is a slice of prices
```

Slices have some special **built-in functions** we can use to work with them. Built-ins are predeclared by the language, so you can use them directly without importing anything. They’re handled specially by the compiler and work across multiple types like arrays, slices, maps, and strings. They are not reserved keywords and as such you can write a function named `len` for example if you want, but will them in the scope of that function los access to the built-in `len`. Built-ins come from the `builtin` Go package, which also interestingly includes value types like `string`, `int`, etc.

`len` gives us the number of items in a slice or array. `cap` gives us the max size `n` of the array, or for a slice the number of elements from the starting index of the slice to the end of the underlying array. This reflects how a slice can grow forward within the array, but not backward as explained earlier.

`lists.go`:
```go
package main

import "fmt"

func main() {
	var productNames [4]string
	productNames = [4]string{"Apple"}
	productNames[2] = "Kiwi"
	fmt.Println(productNames)

	prices := [4]float64{10.99, 9.99, 45.99, 20.0}
	fmt.Println(prices)
	fmt.Println(prices[2])

	featuredPrices := prices[1:]
	featuredPrices[0] = 199.99
	highlightedPrices := featuredPrices[:1]
	fmt.Println(highlightedPrices)
	fmt.Println(prices)
	fmt.Println(len(highlightedPrices), cap(highlightedPrices))
	
	// You can even re-slice on an exsiting slice
	highlightedPrices = highlightedPrices[:3]
	fmt.Println(highlightedPrices)
	fmt.Println(len(highlightedPrices), cap(highlightedPrices))
}
```

Result:
```sh
[Apple  Kiwi ]
[10.99 9.99 45.99 20]
45.99
[199.99] # Highlighted Prices
[10.99 199.99 45.99 20]
1 3
[199.99 45.99 20] # Highlighted Prices grew right
3 3
```
