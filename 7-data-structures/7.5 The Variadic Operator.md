The **variadic operator** `...` is the same as the spread/rest operator in TypeScript. It can be used in function parameters to accept a variable number of arguments, and at call sites to expand (unpack) a slice into individual elements. You can only unpack and slice and not an array (an easy trick to get a slice of the whole array is to do `arr = arr[:]`).

It turns out `append` actually has the signature  `append(slice []T, elems ...Type) []T`. We can append multiple values in one call. We could use this to `append` a whole array or slice to a slice, by unpacking in to the `append` call.

`lists.go`:
```go
package main

import "fmt"

func main() {
	prices := []float64{10.99, 8.99}
	fmt.Println(prices[0:1])
	prices[1] = 9.99

	prices = append(prices, 5.99)
	prices = prices[1:]
	fmt.Println(prices)

	discountPrices := []float64{101.99, 80.99, 20.59}
	prices = append(prices, discountPrices...)
	fmt.Println(prices)
}
```

Result:
```sh
[10.99]
[9.99 5.99]
[9.99 5.99 101.99 80.99 20.59]
```
