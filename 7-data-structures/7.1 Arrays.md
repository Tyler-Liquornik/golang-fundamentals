**Arrays** in go are written with the square brackets *before* the array name, like `[n]arr`, where `n` is the fixed length of the array.

`lists.go`:
```go
package main  
  
import "fmt"  
  
func main() {  
    prices := [4]float64{10.99, 9.99, 45.99, 20.0} 
    fmt.Println(prices)  
}
```


`fmt` can print arrays cleanly, with whitespaces separating entries. Even if we print an array with only default values in its slots it works fine. We can also read and write array values by index, with 0-indexing like most other languages.

`lists.go`:
```go
package main

import "fmt"

func main() {
	var productNames [4]string
	productNames = [4]string{"Apple"}
	prices := [4]float64{10.99, 9.99, 45.99, 20.0}
	fmt.Println(prices)
	productNames[2] = "Kiwi"
	fmt.Println(productNames)
	fmt.Println(prices[2])
}
```

Result:
```sh
[10.99 9.99 45.99 20]
[Apple  Kiwi ] # Apple, "", Kiwi, ""
45.99
```
