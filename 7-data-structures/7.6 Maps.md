Create maps in Go using the `map` keyword with `map[K]V{}`. 

Unlike JavaScript objects, which allow adding new properties dynamically, Go structs have a fixed set of fields and cannot be extended at runtime. Maps, however, remain dynamic and allow new keys to be added.

`maps.go`:
```go
package main

import "fmt"

func main() {
	websites := map[string]string{
		"Google":              "https://google.com",
		"Amazon Web Services": "https://aws.com",
	}
	fmt.Println(websites)
}
```

Result:
```sh
map[Amazon Web Services:https://aws.com Google:https://google.com]
```

Let's practice working with maps. Adding new KV pairs to maps is Python like syntax, add the KV pair as if the key were already there and it will be created. To remove a KV pair, use the built in `delete(m map[K]V, key K)`

`maps.go`:
```go
package main

import "fmt"

func main() {
	websites := map[string]string{
		"Google":              "https://google.com",
		"Amazon Web Services": "https://aws.com",
	}
	fmt.Println(websites)
	fmt.Println(websites["Amazon Web Services"])
	websites["LinkedIn"] = "https://linkedin.com"
	fmt.Println(websites)

	delete(websites, "Google")
	fmt.Println(websites)
}
```

Result:
```sh
map[Amazon Web Services:https://aws.com Google:https://google.com]
https://aws.com
map[Amazon Web Services:https://aws.com Google:https://google.com LinkedIn:https://linkedin.com]
map[Amazon Web Services:https://aws.com LinkedIn:https://linkedin.com]
```
