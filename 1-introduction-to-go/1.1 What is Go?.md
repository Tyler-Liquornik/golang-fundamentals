**Go (Golang)** is an open source programming language developed in 2007 and published in 2009 by Google. 

It has since become a very popular language for number of advantages over other languages:
- Focus on simplicity, clarity, and scalability
	- Inspired by languages like python, in terms of its clean and understandable syntax
- High performance and focus on concurrency
	- Similar to C or C++ for multithreading
- Batteries included
	- Many core features built into the standard library, minimizing need for dependencies
- Static typing
	- Go is a compiled language that is well designed for type safety

Go has since become a favourite for many backend use cases like:
- Networking & APIs
- Microservices
- CLI Tools
