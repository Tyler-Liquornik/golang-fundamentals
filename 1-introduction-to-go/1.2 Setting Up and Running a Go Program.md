Let's first install Go onto our machine and setup an environment. Download the installer from [go.dev](go.dev), and create a Hello World project GoLand:

`app.go`:
```go
package main  
  
import "fmt"  
  
func main() {  
    fmt.Print("Hello World")  
}
```

> Run the code using the go binaries installed, with `go run app.go`. To run a `*.go` file, it must have `package main` and `func main()`.
